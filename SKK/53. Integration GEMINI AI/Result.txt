Berikut beberapa cara untuk membuat koneksi SQL ke PHP:

**Metode 1: Menggunakan MySQLi (direkomendasikan)**

MySQLi (MySQL Improved) adalah ekstensi yang direkomendasikan untuk menghubungkan ke database MySQL.  Ini menawarkan antarmuka prosedural dan berorientasi objek.

```php
<?php

// Data koneksi
$servername = "localhost"; // Nama server database, biasanya localhost
$username = "nama_pengguna"; // Nama pengguna database Anda
$password = "kata_sandi"; // Kata sandi database Anda
$dbname = "nama_database"; // Nama database Anda

// Membuat koneksi
$conn = new mysqli($servername, $username, $password, $dbname);

// Memeriksa koneksi
if ($conn->connect_error) {
  die("Koneksi gagal: " . $conn->connect_error);
}

echo "Koneksi berhasil!";

// Menutup koneksi (penting!)
$conn->close();

?>
```

**Metode 2: Menggunakan PDO (PHP Data Objects)**

PDO adalah ekstensi yang lebih umum yang dapat digunakan untuk terhubung ke berbagai jenis database, tidak hanya MySQL.  Ini menawarkan fleksibilitas yang lebih besar.

```php
<?php

// Data koneksi
$servername = "localhost";
$username = "nama_pengguna";
$password = "kata_sandi";
$dbname = "nama_database";

try {
  // Membuat koneksi
  $conn = new PDO("mysql:host=$servername;dbname=$dbname", $username, $password);
  // Mengatur mode error ke exception
  $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
  echo "Koneksi berhasil!";
} catch(PDOException $e) {
  echo "Koneksi gagal: " . $e->getMessage();
}

// Menutup koneksi (otomatis ketika skrip selesai, tetapi praktik yang baik untuk menutupnya secara eksplisit)
$conn = null;

?>
```

**Penjelasan Kode:**

* **`$servername`**:  Alamat server database. Biasanya "localhost" jika database berada di server yang sama.
* **`$username`**:  Nama pengguna database Anda.
* **`$password`**:  Kata sandi database Anda.
* **`$dbname`**:  Nama database yang ingin Anda hubungkan.
* **`$conn`**:  Variabel yang menyimpan objek koneksi.
* **`mysqli_connect_error()` / `$conn->connect_error`**: Fungsi untuk memeriksa kesalahan koneksi.
* **`die()`**:  Menghentikan eksekusi skrip dan menampilkan pesan error.
* **`$conn->close()` / `$conn = null;`**: Menutup koneksi.

**Langkah-langkah penting:**

1. **Instalasi dan konfigurasi:** Pastikan ekstensi MySQLi atau PDO diaktifkan di server PHP Anda. Anda dapat memeriksa ini dengan membuat file `phpinfo.php` yang berisi `<?php phpinfo(); ?>` dan membukanya di browser.
2. **Data koneksi:** Ganti placeholder `nama_pengguna`, `kata_sandi`, dan `nama_database` dengan kredensial database Anda yang sebenarnya.
3. **Penanganan kesalahan:** Selalu periksa kesalahan koneksi untuk memastikan skrip Anda berfungsi dengan baik.
4. **Keamanan:** Jangan pernah menyimpan kredensial database langsung dalam kode Anda, terutama di lingkungan produksi. Gunakan file konfigurasi terpisah atau variabel lingkungan.


Semoga contoh kode ini membantu Anda membuat koneksi SQL ke PHP. Jika Anda mengalami masalah, berikan detail kesalahan yang Anda terima agar saya dapat membantu Anda lebih lanjut.
